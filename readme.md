
## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/SemenShakhray/comphortel-test.git
cd comphortel-test
```
2. Запустите приложение в контейнере

```bash
docker-compose up
```

3. Запустите миграции

```bash
make install-deps //установка goose при необходимости
make migration-up
```
